Linguagem padrão nos navegadores.

.lenght: conta quantos caracteres há na string, incluindo os espaços.

Prompt: faz perguntas ao usuário.

Strings: sequência de caracteres, letras, numeros e espaços.

Operadores:
=== Igual a
!== Diferente de
and - &&
or - || (conhecido como pipi)
not - !

Arrays:
Loop - Recebe 3 argumentos: declaração da variavél inicial, ate aonde queremos que essa variavél cresça e o que queremos fazer ao final de cada iteração.
For: repetidor de tarefas.

While: Também é um repetidor, ao contrário do FOR, é usado quando não se sabe quantas vezes repetirá a tarefa.

toFixed: limita a quantidade de casas decimais.

Funções: São estruturas que nos possibilita isolar uma determinada lógica e dar um nome a ela,
permitindo que a chamemos em vários pontos do código.

Anônima: permiti passa-la como se fosse um objeto qualquer, que pode atribuir a uma
variavél, independentemente de haver um nome para a função

Console.log(), função que recebe como parâmetro o conteudo da mensagem que passamos.

Document é uma variavél muito importante do JS, ela contém o DOM.

Eventos

É comum adicionar um comportamento ao clique de um elemento, ao usuário passar o mouse em cima,
quando o texto de um input for alterado entre outros eventos. Estas ações são chamadas de eventos,
que serão disparados em determinado momento.

querySelector

Aceita fazer buscas através de um elemento HTML, classe ou ID, utilizando seletores "." "#"(mais especifico para fazer buscas);
document.querySelector(".exemplo")

textContent

É usado para textos, acessa o valor, conteúdo do seu html com o nome passado
Não chamamos apenas a variavél, pois ela retorna a tag. Por isso devemos acessar var.textContent

Form:

action: direciona para outra página;
onsubmit: informe uma função para validar no JS. Ex: "validacao();"
para evitar a atualização automática da página, use return false ou preventDefault;
require: informa o preenchimento obrigatório no campo;
focus: indica o campo obrigatório a ser preenchido;
indexOf: verifica se o caracter existe;
type=email: ja entende que o preenchimento tem que ter @;

Style

O JS também faz atribuições de estilos, porém é melhor deixar essa parte com o CSS, só para saber:
classList para acessar e através do add podemos atribuir algo como "backgroundColor"
Ex: teste.classList.add("backgroundColor");

Um botão criado dentro da tag form tem por padrão enviar os dados para outra página, se a pagina a ser direcionada
não esta definida ele atualiza e apaga os dados preenchidos. Para evitar esse comportamento usamos o
(precisa receber o parametro na função anonima para funcionar) event.preventDefault(),

appendChild - adiciona como filho o elemento passado a ela como parâmetro.

Objetos representam coisas que têm características em comum. É criado com {} separado por virgula, e o ultimo objeto não precisa ser fechado
