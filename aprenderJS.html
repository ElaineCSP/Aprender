O que é JS?

É uma linguagem interpretada.

O que é Ecma script-262?

É JS. Foi criado para tentar unificar a sintaxe e garantir a compatibilidade entre os diversos
navegadores do mercado.

D.O.M

Ele é uma representação de uma pagina HTML no mundo JS e pode ser acessado a partir da variavél
document que representa a raiz da página.

Funções

São estruturas que nos possibilita isolar uma determinada lógica e dar um nome a ela,
permitindo que a chamemos em vários pontos do código.

Função Anônima

A finalidade é a de permitir passa-la como se fosse um objeto qualquer, que pode atribuir a uma
variavél, independentemente de haver um nome para a função.

Eventos

É comum adicionar um comportamento ao clique de um elemento, ao usuário passar o mouse em cima,
quando o texto de um input for alterado entre outros eventos. Estas ações são chamadas de eventos,
que serão disparados em determinado momento.

querySelector

Recebe uma string que é um seletor. É a string que vai dizer se você quer buscar por ID, Classe
ou por tagNme. Ele sempre retorna um elemento.

textContent

É usado para textos, acessa o valor, conteúdo do seu html com o nome passado
