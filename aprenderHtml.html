O html é uma linguagem de marcação, indica o que cada trecho representa.
h1 p são marcações que chamamos de tag e blocos de contrução fundamentais.

METACHARSET-UTF8 - indica a codificação de caracteres da página, acentos, tio etc.

INPUT

Especifica um campo de entrada onde o usuário pode inserir dados. São usados dentro de um elemento form para declarar controles de entrada que permitem aos usuários inserir dados.

AppendChild

Essa função adiciona como filho o elemento passado como parametro

MAIN

Especifica o conteúdo principal de um documento.
Ele não deve conter qualquer conteúdo que seja repetido em documentos com barras laterais, links de navegação, informações de direitos autorais, logotipos de sites e formulários de pesquisas.

*Não deve haver mais de um elemento main em um documento, nem deve ser um descendente de um elemento.
Facilita acesso de busca.

Função anônima

Permite passa-la como se fosse um objeto qualquer que você pode atribuir a uma variável, independente de haver um nome.

DOM representação do HTML para o JS é acessado com document. Usado pelo navegador para desenhar o site.
OBS: Como manipulamos o DOM, alteramos a representação em memória, deixando o HTML intacto.

DOCTYPE html - informa a versão atual que estamos usando em nossa página!

Existe links relativos (relativo ao endereço que estamos visitando no momento) e absolutos (direciona para outro site)
O mais indicado é usar o relativo, pois mesmo que mudemos o nome de nosso site, ele permanecerá intacto, ja usando absoluto
o site não reconhecerá a referência.
Obs: Qualquer link que não comece com http é considerado relativo.
